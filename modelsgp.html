Introduction:

This project comprises of a task in which a personal cv website is to be created. There are in total six web pages; index.html, about.html, social.html, cv.html, modelsgp.html, and contact.html. The index.html is the home page which occurs at the begining of the web page and about.html includes some basic information of the developer and his cv is recorded inside cv.html. Viewers can communicate with the developers through social sites inside social.html and leave any query through contact.html. The report of this project is stored inside the modelsgp.html.
